# 🎒 Animated Signup Page

An interactive and modern **Signup Page** featuring a cute cartoon character powered by **Lottie animations**, **GSAP**, and **sound effects**.
The character walks in with a bag, opens it, and reveals the signup form with smooth and engaging animations.

---

## 🚀 Features

* 🎭 Cartoon animation using **Lottie**
* ⚡ Smooth transitions powered by **GSAP**
* 🎒 Unique bag-opening form reveal animation
* 🔊 Integrated sound effects (zip, click, success, error)
* 📝 Modern signup form with validation
* 👁️ Show/Hide password toggle
* 🎨 Glassmorphism UI design
* 📱 Fully responsive (mobile + desktop)
* 🎉 Success & error animations with character reactions

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Animations & Glassmorphism)**
* **JavaScript (ES6)**
* **Lottie Web**
* **GSAP (GreenSock Animation Platform)**

---

## 🎬 How It Works

1. Cartoon character walks onto the screen 🎭
2. Stops and interacts with the user 👋
3. Opens the bag 🎒
4. Signup form appears with glow effect ✨
5. User fills details and submits 📝
6. Character reacts with success 🎉 or error 😢

---

## 📦 Installation

```bash
git clone https://github.com/official-alimm/animated-signup-page.git

```

Open `index.html` in your browser.
